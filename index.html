Alguns projetos realizados

Sistema BrainQualifica

Sistema de Automação para a empresa farmacêutica Brainfarma - Anápolis. Esse sistema permitia os Analistas de RH a fazer a gestão de 500+ possíveis colaboradores para um novo programa da Empresa "BrainQualifica", fazendo todo o processo em um único sistema (utilizando VBA - Macros), desde a inscrição na vaga até a integração do novo colaborador, tudo em poucos cliques e etapas, dessa forma o time de RH economizou muito tempo de processo. Tempo de Processo reduzido de dois meses para uma semana.

Automação - Análise Empresarial

Ferramenta criada em Python para Análises Empresarial. Criado para a unidade Senai Roberto Mange de Anápolis, esse projeto consiste na Análise dos Colaboradores de um determinado setor levando em consideração as suas habilidades no dia-a-dia de trabalho. Utilizando uma base de dados (Excel) para a absorção dos dados e dessa forma, o sistema consegue fazer diversas análises e relacionar vários indicadores. Fazendo com que os gestores da Empresa consigam criar de forma mais eficaz o PDI de cada colaborador.
